## Problema 10

Clase para números complejos. Todo en un único archivo, les debo la separación en headers y eso.

```
gtheler@tom:~/phd/c++/c++1-10$ make
g++ -g -c complejo.cpp
g++ -o complejo complejo.o
gtheler@tom:~/phd/c++/c++1-10$ ./complejo 
a = 1 + 2i
b = 1 - 2i
+a = 1 + 2i
-a = -1 - 2i
a+b = 2
a*b = 5
a/b = -0.60 - 0.80i
i^2 = -1
-1/i = -1i
gtheler@tom:~/phd/c++/c++1-10$ 
```
